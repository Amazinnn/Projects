# Pixel Snake 贪吃蛇游戏 #

## 简介 ##

基于C语言开发的命令行贪吃蛇游戏，支持多难度级别和动态食物系统。

## 功能特性 ##

- 🎮 经典贪吃蛇操作（WASD控制）
- 📊 4种难度级别（简单/中等/困难/专家）
- ⚡ 可调节移动速度（10-1000ms）
- 🍎 食物系统（增加长度）
- ☠️ 毒药机制（触碰即结束）
- 📈 动态难度成长系统

## 游戏控制 ##

### 主菜单 ###

- **W/S**：选择选项
- **回车**：确认

### 游戏中 ###

- **WASD**：移动方向
- **P**：暂停 | **C**：继续
- **Q**：退出游戏

## 游戏规则 ##

- 吃食物(`@`)：长度+1
- 避免：撞墙、自咬、毒药(`*`)
- 难度越高，毒药越多，食物消失越快

## 系统要求 ##

- Windows系统
- 支持C语言的编译器
